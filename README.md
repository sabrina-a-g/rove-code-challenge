# rove-code-challenge

- Creating a three pane web-page view
- Left Column is a list of locations
- When you click on the location in the left column it should display the location details 
- The center column (can be blank if no location is selected)
- The right column is a continuous ad slideshow
- Each slide displays for 10 seconds then goes to the next slide then loops back to the beginning after the last slide

## How to Run
- clone the repo with 'git clone https://github.com/sabrina-a-g/rove-code-challenge.git'
- run 'npm install'
- run 'npm start'
- copy and paste the "Server running at ..."
